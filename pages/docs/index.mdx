## infinite

`infinite` is both language and library for `infinite-lang` or your custom language.

## infinite-lang

`infinite-lang` is designed for infinitely-extensible programming language.

And purpose of `infinite-lang` is fully-modulized and extensible and also selectable language.

### Main features

- ✅ Fully-declarative lexer and parser
- ⏩ Easy to use, it needs a little bit of parser or compiler knowledge.
- ♻️ Reusable parser declaration

### Features that working in progress

- 🕒 Transformer for AST
- 🕒 Easy-peasy Interpreter declaration
- 🕒 Typescript type for user-defined AST
